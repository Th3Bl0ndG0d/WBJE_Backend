Cylinders

Endpoints for managing cylinders within a job. Accessible roles: service (ROLE_ADMIN) and operator (ROLE_USER).

Get a single cylinder

Returns a specific cylinder by id, scoped to the given job. Accessible roles: service (ROLE_ADMIN), operator (ROLE_USER).

Authorizations:
keycloak
path Parameters
jobId
required
integer <int64>
cylinderId
required
integer <int64>

Responses

Generated server url

http://localhost:8080/jobs/{jobId}/cylinders/{cylinderId}

  • 404

application/json

}
